# Canonical Tool Inventory Template

Status: canonical control asset template  
Date: 2026-04-15  
Owner: TACA with KDGA and QAS support  
Confidentiality: proprietary internal framework; no external publishing approved  

## Purpose

Provide the source-of-truth inventory for approved governed tools and services.

## Usage Rules

- Record every approved tool or service used in governed delivery.
- Use one row per tool or tool configuration when approval scope differs materially.
- Track access scope, write power, and fallback path explicitly.

## Template

```markdown
# Canonical Tool Inventory

| Tool or Service | Owner | Status | Approved Use Case | Client Profile or Scope | Repository or Surface | Access Scope | Write or Deploy Power | Credential Rule | Data Exposure Risk | Fallback Path | Last Review Date | Notes |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| | | | | | | | | | | | | |
```
