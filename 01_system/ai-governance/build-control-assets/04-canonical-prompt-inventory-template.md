# Canonical Prompt Inventory Template

Status: canonical control asset template  
Date: 2026-04-15  
Owner: PCA with KDGA support  
Confidentiality: proprietary internal framework; no external publishing approved  

## Purpose

Provide the source-of-truth inventory for approved governed prompts.

## Usage Rules

- Record every approved governed prompt instance here.
- Do not treat chat history as the prompt registry.
- Use one row per prompt instance, not one row per prompt concept.
