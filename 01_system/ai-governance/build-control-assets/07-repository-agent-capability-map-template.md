Status: canonical control asset template

Owner: PRGA with MOA, RCA, and PMA support  
Confidentiality: proprietary internal framework; no external publishing approved  

## Purpose

Provide the routing map used to determine which agents are relevant and capable for each repository, module, and technical surface.

## Usage Rules

- Keep one map per governed repository or tightly related repository group.
- Update when new specialists are added, retired, or materially re-scoped.
- Use this map during build intake and handoff assessment.
